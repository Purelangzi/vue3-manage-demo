# vue3-ts-demo

电商后台，只做了登录注册，首页Echarts，权限页的用户列表，动态路由。用到的技术vue3+ts+pinia,Echarts,flex,


## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
